This is a project to create a raspberry pi walkman with an eink screen 

I am using this code as a base, but with modification to fit my needs and the different hardware:
https://hackaday.io/project/196631-epipod

This is the screen i'm using (and the code for the screen):
https://www.waveshare.com/wiki/2.9inch_Touch_e-Paper_HAT

And the hardware:
Raspberry pi Zero W
UPS Lite V1.2 UPS Power Hat Board With 1000mAh
And some physical buttons
